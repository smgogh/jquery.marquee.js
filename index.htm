概述

jQuery Marquee 转换列表元素 (<ul /> 或 <ol />) 到 ESPN-style scrolling marquee. 消息会从顶部或底部滚动 (基于 yScroll 选项) 更长的消息向左滚动以显示完整的消息。

要求

为了使用Marquee插件，需要具备以下条件：

  • jQuery v1.2.6 (或更高版本)
  • jquery.marquee.js 插件
*这个插件可以在1.2家庭旧版本的jQuery的工作。如果您尝试使用这种使用旧版本的jQuery，你将需要包括jquery.dimensions.js插件（如jQuery的V1.2.6插件包含在jQuery的核心jquery.dimensions.js）

用法

要转换您的 <ul /> (or <ol /> 元素融入选取框的元素，只需调用jQuery插件使用如下语法： 

$("ul").marquee([options]);

参数

选项

此参数是可选的，并允许您自定义用于插件的每个实例的设置。有关所有可用选项的列表，请参阅 选项 部分。

现在我们已经到窗口小部件的引用，我们可以调用任何公共API调用。

公共API

$("#id").marquee("update")

这将更新字幕的缓存，这是在动态更新<ul /> 元素内容后，以显示新的元素。

$("#id").marquee("pause")

暂停当前选取框，以防止其滚动。

$("#id").marquee("resume")

恢复选取框所以它再次开始滚动。

$("#id").marquee("object")

返回到内部 $.Marquee 对象的引用。这是唯一的高级用法。 检查使用了 $.Marquee 对象的源代码。

选项

若干选项可用于定制 Marquee 小部件的外观和感觉

{
      yScroll: "top"              // 初始滚动方向 (还可以是"top" 或 "bottom")
    , showSpeed: 850              // 初始下拉速度
	, scrollSpeed: 12             // 滚动速度
	, pauseSpeed: 5000            // 滚动完到下一条的间隔时间
	, pauseOnHover: true          // 鼠标滑向文字时是否停止滚动
    , loop: -1                    // 设置循环滚动次数 （-1为无限循环）
	, fxEasingShow: "swing"       // 缓冲效果
    , fxEasingScroll: "linear"    // 缓冲效果
    , cssShowing: "marquee-showing"//定义class

    // event handlers
    , init: null                // 初始调用函数
	, beforeshow: null          // 滚动前回调函数
	, show: null                // 当新的滚动内容显示时回调函数
	, aftershow: null           // 滚动完了回调函数
}

入门

我们需要做的第一件事就是加载所需的JavaScript库和小部件所使用的CSS样式表：

<!---// load jQuery and the jQuery Marquee Plug-in //--->
<script type="text/javascript" src="//cdn.bootcss.com/jquery/1.3.2/jquery.min.js"></script>
<script type="text/javascript" src="./lib/jquery.marquee.js"></script>

<!---// load the Marquee CSS stylesheet //--->
<link type="text/css" href="./css/jquery.marquee.css" rel="stylesheet" media="all" />

下一步是实际创建一个 Marquee 实例。你要确保初始化后，所有必要的DOM元素可用，这使得 document.ready 事件很好的初始化小部件。 

<script type="text/javascript">
$(document).ready(function (){
  $("#marquee").marquee();
});
</script>
